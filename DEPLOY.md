# Инструкции по развертыванию шахматного приложения

## Варианты хостинга

### 1. Heroku (Рекомендуется)

**Преимущества:** Простота развертывания, автоматическое масштабирование, встроенная база данных

**Шаги развертывания:**

1. Создайте аккаунт на [Heroku](https://heroku.com)
2. Установите [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)
3. Выполните команды:

```bash
# Инициализация git репозитория
git init
git add .
git commit -m "Initial commit"

# Создание приложения на Heroku
heroku create your-chess-app-name

# Добавление PostgreSQL базы данных
heroku addons:create heroku-postgresql:mini

# Установка переменных окружения
heroku config:set SECRET_KEY=$(python -c 'import secrets; print(secrets.token_hex(16))')

# Развертывание
git push heroku main

# Инициализация базы данных
heroku run python -c "from app import app, db; app.app_context().push(); db.create_all()"
```

**URL:** https://your-chess-app-name.herokuapp.com

### 2. Railway

**Преимущества:** Современная платформа, простое развертывание из GitHub

**Шаги развертывания:**

1. Создайте аккаунт на [Railway](https://railway.app)
2. Подключите GitHub репозиторий
3. Railway автоматически обнаружит Python приложение
4. Добавьте PostgreSQL базу данных в проекте
5. Установите переменную окружения `SECRET_KEY`

### 3. Render

**Преимущества:** Бесплатный план, автоматическое развертывание

**Шаги развертывания:**

1. Создайте аккаунт на [Render](https://render.com)
2. Создайте новый Web Service из GitHub репозитория
3. Render автоматически использует `render.yaml` конфигурацию
4. База данных PostgreSQL будет создана автоматически

### 4. PythonAnywhere

**Преимущества:** Специализируется на Python, простая настройка

**Шаги развертывания:**

1. Создайте аккаунт на [PythonAnywhere](https://pythonanywhere.com)
2. Загрузите файлы через веб-интерфейс или git
3. Настройте веб-приложение в панели управления
4. Установите зависимости: `pip3.11 install --user -r requirements.txt`

### 5. DigitalOcean App Platform

**Преимущества:** Профессиональная платформа, хорошая производительность

**Шаги развертывания:**

1. Создайте аккаунт на [DigitalOcean](https://digitalocean.com)
2. Создайте новое приложение из GitHub репозитория
3. Добавьте управляемую базу данных PostgreSQL
4. Настройте переменные окружения

## Настройка переменных окружения

Для всех платформ установите следующие переменные:

- `SECRET_KEY` - секретный ключ для Flask (генерируется автоматически)
- `DATABASE_URL` - URL базы данных (обычно предоставляется хостингом)

## Важные замечания

1. **WebSocket поддержка:** Убедитесь, что хостинг поддерживает WebSocket для Socket.IO
2. **База данных:** Приложение автоматически создаст таблицы при первом запуске
3. **HTTPS:** Большинство хостингов предоставляют HTTPS автоматически
4. **Домен:** Можете настроить собственный домен в настройках хостинга

## Рекомендуемый порядок действий

1. **Heroku** - для быстрого развертывания и тестирования
2. **Railway** - для современного опыта разработки
3. **Render** - для бесплатного хостинга с хорошими возможностями
4. **DigitalOcean** - для продакшн приложений с высокой нагрузкой

## Мониторинг и логи

После развертывания вы можете просматривать логи:

- **Heroku:** `heroku logs --tail`
- **Railway:** Встроенная панель логов
- **Render:** Вкладка "Logs" в панели управления

## Обновление приложения

Для обновления просто отправьте изменения в git репозиторий:

```bash
git add .
git commit -m "Update description"
git push heroku main  # для Heroku
git push origin main  # для других платформ с автодеплоем
```
