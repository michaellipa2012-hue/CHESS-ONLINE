# Шахматы Онлайн

Веб-приложение для игры в шахматы в реальном времени с регистрацией, авторизацией и возможностью выбора цвета фигур.

## Возможности

- 🔐 Регистрация и авторизация пользователей
- 🎯 Игра в шахматы в реальном времени
- 👥 Поиск соперников онлайн
- ⚪⚫ Выбор цвета фигур (белые, черные, случайно)
- 🎨 Красивый сине-голубой дизайн
- 📱 Адаптивный интерфейс

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Технологии

- **Backend**: Flask, Flask-SocketIO, SQLAlchemy
- **Frontend**: HTML, CSS, JavaScript, Socket.IO
- **База данных**: SQLite
- **Реальное время**: WebSocket через Socket.IO

## Как играть

1. Зарегистрируйтесь или войдите в систему
2. В лобби выберите желаемый цвет фигур
3. Нажмите "Найти игру" для поиска соперника
4. Играйте в шахматы в реальном времени!

## Особенности реализации

- Использование WebSocket для мгновенной синхронизации ходов
- Сохранение состояния игр в базе данных
- Система комнат для изоляции игровых сессий
- Проверка валидности ходов на стороне сервера
- Красивый градиентный дизайн в сине-голубых тонах
